# SnapQR by JWS

GitHub Pages-ready QR code maker for Johnson Website Solutions LLC.

## What is included

- 50-tool QR menu
- Static QR payloads for URL, Wi-Fi, vCard, phone, SMS, email, maps, text, coupon, event, product, payment, reviews, Facebook, and YouTube
- Local demo dynamic QR records
- Local browser account/session demo
- Owner/admin dashboard
- PNG, SVG, print/PDF workflow
- QR library with saved records
- JWS branding and website link

## Owner setup

Open the app and use the owner email shown in the login panel. On the first login, create a passcode for that browser. It is hashed and stored locally in browser storage.

## Important note

This is a static GitHub Pages build. Local demo accounts are good for testing the product flow, but production customer accounts and truly editable dynamic QR links require a backend database/API.
